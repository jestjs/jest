Jest コミュニティ

Jest 周辺のコミュニティは、テストの経験をより良いものにするために懸命に努力しています。

jest-community は、新しい GitHub organization です。この organization では、高品質の Jest 拡張が Jest のメンテナやコラボレータたちによって収集されています。 私たちお気に入りのプロジェクトをいくつか挙げてみましょう。

  • vscode-jest
  • jest-extended
  • eslint-plugin-jest
  • awesome-jest

一つの organisation の下にコミュニティのプロジェクトを置くというのは、Jest の新しいアイデア/技術およびアプローチを試すための素晴らしい方法です。 コミュニティからのコントリビューションや速いペースでの独立したコントリビューションの公開を推奨します。

jest-community org は、Jest に関係する素晴らしいプロジェクトと関連資料をリストアップした awesome-jest リストもメンテナンスしています。ここには、jest-community org には含まれていないすべてのプロジェクトも集められています。

もしあなたがみんなと共有すべき素晴らしいものを持っていたら、気軽に私たちに教えてください！ 私たちはあなたのプロジェクトを喜んで awesome-jest リストで共有させてもらいます (ここから PR を送ってください)。あるいは、プロジェクトを jest-community org に移動させたい場合は、jest-community org のオーナーの誰かに連絡してください。
