Mais Recursos

Agora você deve ter uma boa ideia de como Jest pode facilitar testar suas aplicações. Se você está interessado em aprender mais, aqui tem algumas coisas relacionadas que você pode querer ver.

Navegar na documentação #

  • Saiba mais sobre Teste de Snapshot, Funções Mock e mais em nossos guias aprofundados.
  • Migre os seus testes existentes para Jest seguindo nosso guia de migração.
  • Aprenda como configurar Jest.
  • Confira a API de Referência completa.
  • Solucionar problemas com Jest.

Aprender com exemplos #

Você vai encontrar um número de exemplos de casos de teste na pasta examples no GitHub. Você também pode aprender com os excelentes testes usados pelos projetos React, Relay e React Native.

Participe da Comunidade #

Faça perguntas e encontre respostas de outros usuários como você de Jest. Reactiflux é um chat Discord onde acontece muita discussão sobre Jest. Confira o canal de #jest.

Siga a conta no Twitter do Jest e blog para descobrir o que está acontecendo no mundo de Jest.
